<h1>Small Python Projects</h1>
<p>Small Python project that execute mainly on the terminal. I used them to learn Python fundamentals.</p>
<p>Technologies and concepts: Python, loops, conditionals, OOP, file reading, input taking and formatting</p>

<h2>simple-calculator.py</h2>
<p>The simple-calculator.py has methods to add, subtract, multiply and divide floats entered by an user and provide the result of the calculation. Concepts: Loops, Methods </p>

<h2>reverse-name.py</h2>
<p>The user enters a name and the program spits out the name in reverse. Concepts: Loops, string formatting</p>

<h2>guess-the-number.py</h2>
<p>The computer produces a random number that the user needs to guess in 10 attempts. Concepts: random module, loops, try-catch blocks(exceptions), user input</p>

<h2>memory-sequence-game.py</h2>
<p>Presents the user with a sequence of integers (waiting a second in between the integers) that he or she has to remember and type back on the console. Concepts: loops, random module, os module, time module</p>

<h2>task-manager.py</h2>
<p>Application that gives the user the choice to add tasks, view them, mark them as complete, and delete them. Concepts: Lists, Methods, Loops</p>

<h2>rock-paper-scissors</h2>
<p>Simple rock-paper-scissors game where the user makes a choice and the computer makes a random choice and the winner is decided by the rules. Concepts: methods, exceptions, loops</p>

<h2>word-counter.py</h2>
<p>The user inputs a text and the console prints a summary that includes:</p>
<ul> 
  <li>word count</li>
  <li>character count(spaces included)</li>
  <li>character count(spaces not included)</li>
  <li>sentence count</li>
  <li>average words per sentence</li>
  <li>average characters per word</li>
</ul>
